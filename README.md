# 🚴 Bike Buyers Sales Analysis (Excel)

## Tools Used
- Microsoft Excel
- PivotTables
- Data Cleaning
- Dashboard Design

## Project Overview
This project analyzes customer demographic data to understand purchasing behavior for bike sales. The goal was to identify trends based on income, age, and commute distance.

## Key Insights
- Higher income groups had higher purchase rates.
- Commute distance influenced buying behavior.
- Age group segmentation showed clear buying patterns.

## Dashboard Features
- Purchase Analysis by Income
- Demographic Breakdown
- Interactive Filters
